                                   摘要
   本文研究具有线性动力学的最优控制终端问题的数值解。研究对象为常系数线
性控制系统


         ẋ(t) = Ax(t) + Bu(t),   x(0) = x0 ,   u(t) ∈ U,   t ∈ [0, T ],

     其中 A 与 B 为给定常数矩阵，U 为非空紧凸控制集。论文围绕两个终端型问题
展开：一是二次终端误差最小化问题，二是带仿射终端目标集约束的线性终端泛函
最小化问题。首先建立线性系统解公式、允许控制集、可达集和支撑函数等基础概
念；其次在统一的最小化框架下引入最大化形式伴随变量 ψ = −p，将最大值条件
写成支撑函数上的线性最大化问题；然后利用线性伴随方程的结构，用终端伴随向
量 q = ψ(T ) 或乘子 λ 参数化整条伴随轨线、控制函数和状态轨线；最后把原无限
维控制函数优化问题转化为有限维残差最小二乘问题，并给出离散递推、有限差分
Jacobian、Armijo 线搜索、高斯–牛顿迭代和光滑化处理等可实现的数值算法。数值
实验部分通过一维系统、二维终端跟踪、仿射终端约束和非光滑控制集光滑化对比
等算例，验证算法在终端误差、终端约束残差、目标函数下降和控制曲线稳定性方
面的效果。

1.1 研究背景与意义

   最优控制理论研究如何在给定动力系统、约束条件和性能指标下选择控制函数，
使系统运动达到某种最优效果 [1, 第 13–27 页][3, 第 76–91 页][4, 第 8–15 页]。在线性
动力学系统中，状态方程具有较清晰的解析结构，矩阵指数、可达集、支撑函数和
伴随方程都可以得到相对明确的表达，因此线性系统既是最优控制理论的重要基础，
也是构造数值算法的重要对象。
   终端型最优控制问题关注系统在终端时刻的状态表现。例如，使终端状态尽可
能接近目标点，或者要求终端状态落入指定目标集，同时使某个终端性能指标最优。
第 1 章绪论                                                   5


这类问题广泛出现在轨迹规划、导航控制、经济过程控制、工程系统调节和资源优
化等问题中。
   本文的研究意义在于：利用线性动力学和伴随方程的结构特点，将原本关于控
制函数的无限维优化问题转化为关于有限维参数的优化问题，并进一步构造可计算
的数值求解流程 [2, 第 180–190 页][6, 第 5–7 页]。这样既能体现庞特里亚金最大值原
理的理论作用，也能为终端型最优控制问题提供实际可实现的计算方法。


1.2 国内外研究现状

   与本文相关的研究可以从最优性条件、线性系统几何结构、数值求解方法和本
文定位四个方面理解。
   第一，庞特里亚金最大值原理为最优控制问题提供了伴随方程、Hamilton 函数
最大化条件和横截条件等必要条件 [1, 第 23–27 页][3, 第 336–371 页][5, 第 119–158
页]。后续教材和专著进一步系统化了变分法、正常与异常极值、终端约束和法锥条
件等内容 [7, 8, 9, 10]。这些理论说明，终端型问题通常不能只看状态方程本身，还必
须同时处理伴随变量和终端横截条件。
   第二，在线性控制系统中，状态对控制的依赖是线性的，可达集在紧凸控制集
作用下具有凸几何结构 [2, 第 15–59 页][3, 第 76–91 页]。支撑函数方法可以把控制集
上的线性最大化写成 cU (η) = maxu∈U hη, ui，从而把最大值条件与可达集的支撑超平
面联系起来 [4, 第 29–37 页][6, 第 3–4 页]。这一路线特别适合分析终端误差最小化和
仿射终端约束下的线性泛函优化。
   第三，数值最优控制方法大体可分为直接法和间接法。直接法把控制或状态轨线
离散化后转化为非线性规划问题，具有实现稳健、便于处理复杂约束的特点 [11, 12]；
间接法先利用最大值原理得到边值问题或低维参数方程，通常维数较低，但对初值、
非光滑性和切换结构更敏感 [2, 第 180–190 页][6, 第 3–10 页]。有限差分 Jacobian、线
搜索、高斯–牛顿和阻尼迭代属于常用的数值优化工具 [13, 14]，可用于求解由间接
法得到的有限维残差方程。
   本文采用间接法与有限维参数优化相结合的路线。与直接把每个时间节点控制
值作为优化变量的做法不同，本文先由最大值原理确定控制结构，再利用线性伴随
方程把未知量压缩为终端伴随向量 q 或乘子 λ。这样既保留了最大值原理的理论解
释，又能形成可复现的有限维最小二乘数值算法。为避免只从单一算法角度解释数
值结果，实验部分还加入一个直接离散优化对比算例，用同一 Euler 离散格式和同一
盒约束检验终端目标的可达性及间接参数化方法的适用边界。

推导均以该常系数线性系统为基础。


2.2 矩阵指数与系统解公式

  对于常系数线性系统：


                  ẋ(t) = Ax(t) + Bu(t),             x(0) = x0

  齐次系统 ẋ(t) = Ax(t) 的解由矩阵指数 eAt 给出 [2, 第 15–31 页][3, 第 76–91 页]。
由常数变易公式，受控系统的解为：

                                         ∫ t
                             At
                  x(t) = e x0 +                eA(t−s) Bu(s) ds
                                          0

  因此终端状态满足：

                                         ∫ T
                 x(T ) = e   AT
                                  x0 +         eA(T −s) Bu(s) ds
                                          0

  该公式说明，在常系数线性系统中，终端状态由初始状态和控制函数共同决定，
并且控制函数以线性方式影响终端状态。这是后续讨论可达集、支撑函数和有限维
参数化的基础。


2.3 允许控制集与可达集

  设控制约束集为：


                                    U ⊂ Rm

  允许控制集合定义为满足控制约束的本质有界可测函数集合，记为：


                             U = L∞ ([0, T ]; U )

  等价地，对任意允许控制，有：


                         u(·) ∈ L∞ ([0, T ]; U )

  终端时刻 T 的可达集定义为：

                        X(T ) = {x(T ; u) : u(·) ∈ U}

  由线性系统解公式可知：

                                   {∫ T                                        }
                                               A(T −s)
           X(T ) = e   AT

  当 U 为非空紧凸集时，控制对终端状态的作用是线性积分映射，因此可达集具
有凸性；在本文采用的有限维常系数系统和有界控制设定下，可达集具有适合投影
和支撑超平面分析的闭凸结构 [3, 第 76–91 页][2, 第 32–40 页]。终端型最优控制问题
可以理解为在可达集 X(T ) 上优化某个终端函数，或在可达集与终端目标集的交集
中寻找最优终端点。


2.4 支撑函数

  对非空紧凸集 U ⊂ Rm ，定义其支撑函数 [4, 第 29–37 页][6, 第 3–4 页]：


                        cU (η) = maxhη, ui,                η ∈ Rm
                                     u∈U

  其中 h·, ·i 表示欧氏内积。由于 U 非空紧，最大值可以取到。
  支撑函数用于描述控制约束集上的线性最大化问题 [6, 第 3–4 页]。在最大值原
理中，最优控制通常满足：


                            u∗ (t) ∈ arg maxhB ⊤ ψ(t), ui
                                           u∈U

  因此有：


                       hB ⊤ ψ(t), u∗ (t)i = cU (B ⊤ ψ(t))

  若 U 为光滑严格凸集，则 cU 可微，最优控制可以写为：


                              u∗ (t) = ∇cU (B ⊤ ψ(t))

  若 U 为区间、盒约束或多面体，则 cU 一般不可微，最优控制可能出现开关结构。
此时可采用次梯度、多值最大化选择或光滑化近似。

 2. 对当前 qk ，计算 ψm (qk )、ηm (qk ) 和 um (qk )。

 3. 由 xm+1 = xm + τ (Axm + Bum ) 正向递推，得到 xM (qk )。

 4. 计算残差 F (qk ) = qk + xM (qk ) − xT 和目标 Θ1 (qk ) = 21 kF (qk )k2 。

 5. 若 kF (qk )k ≤ εR ，停止。

 6. 用有限差分计算 JF (qk )，并取梯度方向或高斯–牛顿方向 dk 。

 7. 用 Armijo 线搜索确定步长 ρk ，令 qk+1 = qk + ρk dk 。

 8. 若 kJF (qk )⊤ F (qk )k ≤ εg 或 k = Kmax ，停止；否则令 k ← k + 1 并返回步骤 2。

  输出：


           q∗,    {u∗m }M −1
                        m=0 ,   {x∗m }M
                                      m=0 ,   kF (q ∗ )k,   eT = kx∗M − xT k.


4.9 算法 2：仿射终端目标集问题

  输入：


                 A, B, x0 , U, T, M, a, C, d, λ0 , h, εR , εg , Kmax .

  计算步骤如下。

 1. 设置 τ = T /M ，取初值 λ0 ，令 k = 0。

 2. 计算 q(λk ) = −a − C ⊤ λk 。

 3. 由 q(λk ) 计算 ψm (λk )、ηm (λk ) 和 um (λk )。

 4. 正向递推状态方程，得到 xM (λk )。

 5. 计算残差 G(λk ) = CxM (λk ) − d 和目标 Θ2 (λk ) = 12 kG(λk )k2 。
第 5 章数值实验与结果分析                                                                      25


 6. 若 kG(λk )k ≤ εR ，停止。

 7. 用有限差分计算 JG (λk )，并取梯度方向或高斯–牛顿方向 dk 。

 8. 用 Armijo 线搜索确定步长 ρk ，令 λk+1 = λk + ρk dk 。

 9. 若 kJG (λk )⊤ G(λk )k ≤ εg 或 k = Kmax ，停止；否则令 k ← k + 1 并返回步骤 2。

  输出：



     λ∗ ,   q(λ∗ ),   {u∗m }M −1
                            m=0 ,   {x∗m }M
                                          m=0 ,    rT = kG(λ∗ )k,   J2 = a⊤ x∗M .


4.10 本章小结

  本章把最大值原理给出的连续最优性条件转化为可计算的数值算法。核心步骤
是用 q 或 λ 生成伴随轨线和控制序列，再通过状态递推得到终端状态，最后建立有
限维残差最小二乘问题。有限差分 Jacobian、Armijo 线搜索、高斯–牛顿方向和光滑
化控制共同构成本文数值解部分的具体实现框架。

6.1 总结

  本文围绕具有线性动力学的最优控制终端问题的数值解展开研究。首先建立了
常系数线性控制系统、允许控制集、可达集、支撑函数以及两个终端型最优控制问题
的数学模型；然后在统一的最小化框架下，引入最大化形式伴随变量 ψ = −p，将最
大值条件表示为支撑函数上的最大化问题；进一步利用线性伴随方程的结构，将未知
控制函数转化为有限维伴随终端参数；最后针对二次终端误差问题和带仿射终端目
标集约束的线性终端泛函问题，构造了离散递推、残差最小二乘、有限差分 Jacobian、
Armijo 线搜索和高斯–牛顿方向组成的数值求解流程。
  主要结论如下：

 1. 对线性动力学系统，伴随方程具有线性结构，整条伴随轨线可由终端向量 q =
   ψ(T ) 决定。

 2. 通过引入 ψ = −p，最小化问题中的控制条件可以统一写成支撑函数的最大化
   形式。

 3. 二次终端误差问题可通过伴随终端条件转化为关于 q 的有限维残差最小二乘问
   题。

 4. 带仿射终端目标集的线性终端泛函问题可通过乘子 λ 进行有限维参数化，并以
   终端约束残差作为数值求解对象。

 5. 有限差分 Jacobian、Armijo 线搜索和高斯–牛顿方向构成了可直接实现的参数
   迭代方法。

 6. 对非光滑控制集，可通过光滑化支撑函数改善切换点附近的数值稳定性。

 7. 数值实验表明，应同时报告有限维残差和实际终端误差，特别是在光滑化参数
   较大或直接 bang-bang 迭代时，二者可能呈现不同含义。

  后续研究可以从以下方向展开：

 1. 将线性动力学推广到非线性动力学系统。

 2. 考虑含状态约束的终端型最优控制问题。

 3. 研究随机扰动或不确定参数下的终端控制问题。
